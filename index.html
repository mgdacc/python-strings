Strings (Cadenas de Texto)

Creación: Comillas Simples

Forma más común de crear un string.

mi_string = 'Hola, mundo'

Creación: Comillas Dobles

Útil si el texto contiene comillas simples.

mi_string = "I'm a developer"

Creación: Comillas Triples

Permite crear strings de múltiples líneas.

texto_largo = """Esto es un string
que ocupa varias líneas."""

Slicing: Acceso por Índice

Obtiene un carácter por su posición (empieza en 0).

s = "Python"
print(s[0]) # 'P'
5 / 22

Slicing: Índice Negativo

Accede a los caracteres desde el final (-1 es el último).

s = "Python"
print(s[-1]) # 'n'
6 / 22

Slicing: Rebanado [x:y]

Obtiene una subcadena desde el índice X hasta el Y-1.

s = "Python"
print(s[1:4]) # 'yth'
7 / 22

Método: .upper()

Convierte todo el string a mayúsculas.

print("Hola".upper()) # 'HOLA'
8 / 22

Método: .lower()

Convierte todo el string a minúsculas.

print("ADIÓS".lower()) # 'adiós'
9 / 22

Método: .strip()

Elimina espacios en blanco al inicio y al final.

print("  Hola  ".strip()) # 'Hola'
10 / 22

Método: .replace()

Reemplaza una subcadena por otra.

print("Hola".replace('o', 'a')) # 'Hala'
11 / 22

Método: .split()

Divide el string en una lista usando un separador.

print("a,b,c".split(',')) # ['a', 'b', 'c']
12 / 22

Método: .join()

Une los elementos de una lista en un solo string.

print("-".join(['a', 'b', 'c'])) # 'a-b-c'
13 / 22

Método: .startswith()

Verifica si el string comienza con un prefijo.

print("Python".startswith('Py')) # True
14 / 22

Método: .endswith()

Verifica si el string termina con un sufijo.

print("Python".endswith('on')) # True
15 / 22

Método: .find()

Devuelve el índice de la primera aparición de una subcadena.

print("Python".find('th')) # 2
16 / 22

Método: .count()

Cuenta cuántas veces aparece una subcadena.

print("banana".count('a')) # 3
17 / 22

Método: .capitalize()

Pone en mayúscula solo la primera letra del string.

print("hola mundo".capitalize()) # 'Hola mundo'
18 / 22

Método: .title()

Pone en mayúscula la primera letra de cada palabra.

print("hola mundo".title()) # 'Hola Mundo'
19 / 22

Formateo: f-Strings

La forma más moderna y recomendada (Python 3.6+).

nombre = "Ana"
print(f"Hola, {nombre}")
20 / 22

Formateo: .format()

Flexible y compatible con versiones anteriores de Python 3.

nombre = "Luis"
print("Hola, {}".format(nombre))
21 / 22

Formateo: Operador %

Estilo antiguo, heredado de C. No se recomienda.

nombre = "Carlos"
print("Hola, %s" % nombre)
22 / 22
